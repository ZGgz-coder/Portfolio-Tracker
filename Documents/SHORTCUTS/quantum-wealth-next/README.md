# Quantum Wealth Next Prototype

Structural prototype for Quantum Wealth (P0):
- Next.js App Router + TypeScript
- 6 tab routes: Pulse, Liquid, Alpha, Frontier, Log, Copilot
- Reusable UI engine: QuantumCard, QuantumGlow, PrivacyNumber, SourceBadge
- Charts with real mock variation (sparkline, bars, scenario line)
- Global privacy mode and per-number privacy toggles
- Zustand normalized mock-first state
- Framer Motion transitions

## Run

```bash
npm install
npm run dev
```

## Validate

```bash
npm run test
npm run build
```
